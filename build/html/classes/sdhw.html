1: 31 3: 1, 4, 6, 8, A uniform cylinder is on an incline with slope α. It is held barely out of contact with the ramp and has an initial angular velocity \(\omega_0\) oriented to propel the cylinder up the ramp. The coefficient of friction for both static and dynamic friction is μ. (It has mass m, radius r).

a) If it is set down on the the ramp, and initially slips, how far up the ramp will it move before it begins rolling without slipping?

b) Once it starts rolling without slipping, how far up the ramp will it move before it stops moving up and reverses course.
